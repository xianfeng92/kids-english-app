# 音频文件获取指南

由于网络限制，推荐以下几种方式获取音频文件：

## 方案一：使用 Forvo（推荐，真人发音，免费）

1. 访问 https://forvo.com/languages/en/
2. 在搜索框输入单词（如 "apple"）
3. 点击播放图标听发音
4. 点击"下载"图标（箭头向下）保存 mp3
5. 将文件重命名为指定名称

## 方案二：使用在线 TTS 工具

### ttsmp3.com（免费，无需注册）
1. 访问 https://ttsmp3.com/
2. 选择语言：English
3. 选择语音：任何带 US 或 UK 标志的
4. 输入单词
5. 点击 "Read text" 播放
6. 点击 "Download" 下载
7. 重命名文件

## 方案三：使用 Edge 浏览器的朗读功能

1. 用 Edge 浏览器打开 https://www.microsoft.com/en-us/ai
2. 选中你想要发音的单词
3. 右键选择"朗读"
4. 同时打开录音软件（如 Windows 录音机）录制
5. 保存为 mp3
